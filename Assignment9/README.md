
# ReactJS Website

This website demonstrates the use of ReactJS applied on a 5 page website.
It consists of a home, login, menu, contact and about page.
This website showcases the use of ReactJS and connection with nodejs. 

# ReactJS Components Used:

5 page website connected using React Router

Login Page communication with nodejs achieved and user is able to login successfully

React Map and Card structure applied on webpage

User is created through API request and is authenticated using React Login Page.

Email and password validations present in Nodejs are applied on React Login Form
